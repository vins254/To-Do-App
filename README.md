####To-Do List App

A simple and interactive To-Do List App built using HTML, CSS and JavaScript.
This beginner-friendly project helps users manage daily tasks by adding, marking as completed, and deleting items.
It also demonstrates practical front-end skills such as DOM manipulation, event handling, and browser storage.


####Project Description

This To-Do List App allows users to:

    -Add new tasks

    -Mark tasks as completed

    -Remove tasks

    -Store tasks in the browser using localStorage

This project is useful for learning:

    -How to manipulate the DOM

    -Basic front-end development

    -Saving persistent data without a database


####Technologies Used

    -HTML5

    -CSS3

    -JavaScript (ES6)
